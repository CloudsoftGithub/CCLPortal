1. Advanced ICT Training

Advanced ICT Training

Pre-requisite

All candidates are expected to have acquired a sound knowledge on any of the following listing:
  1. C or C++
  2. Java fundamentals (both theory and practical)
  3. Usage of IDEs in java programming,
  4. MySQL/PostgreSQL database
  5. Embedded SQL
  6. Software engineering concepts
  7. Cloud computing

Expectations:

At the end of this training, attendees are expected to:

  1. Acquire sound knowledge on software development with Java programming language for both Java desktop application as well as Java Web Applications.
  2. Develop Java application with GUI interface
  3. Develop a holistic CRUD desktop application
  4. Develop a holistic CRUD web application
  5. Install and configure JDBC
  6. Design database and integrate with Java application
  7. Exporting the database with command prompt into a .sql dump_file from the host machine
  8. Importing the .sql dump_file into the target machine
  9. Know how to package the Java desktop application and create an installable java file (Setup.exe)
  10. Host a web app on AWS Cloud
  11. Know the cloud architecture
  12. Perform cloud migration from on-premise data center to cloud
  13. Know the various types of data storages in AWS
  14. Provision cloud instances (servers) and configure them
  15. Provision AWS Microsoft2003 server, install the Microsoft IIS web server, install the Apache tomcat server, install the MySQL database server, install the IIS connector and host a web app on the provisioned server
  16. Create an author scaling group, disaster recovery, etc.
