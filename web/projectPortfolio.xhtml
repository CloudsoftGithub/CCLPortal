Project Name: Electronic Property Solution Software(Epropsol)

Description of the Project:
E-Propsol soft is an acronym for Electronic Property Solution Software, it is a software artifact designed and directed towards finding solutions to property management where all tenancy activities are automated. Which include house registration, owner registration, unit registration, tenants registration, availability checking, lease transact, lease termination, expiry date checking, reminder, lease renewal, balance clearance and reports. E-propsol can be implemented as stand-alone, network-based systems or cloud solution each of which were explained fully in this letter. The software is highly customizable to adhere with the settings of different types of building structures such as plaza, shopping complex, shop, single rooms, room and parlor, two-bedroom-flat, three-bedroom-flat, etc. It is designed to automatically handle all sort of activities carried out during tenancy where all properties belonging to the agency is to be registered and ID will be generated automatically by the system. The system tracks all required information of the properties and is capable of producing required information when needed. This includes whether or not amount paid has expired, checking availability of a given property, accounting information for the property, etc.
